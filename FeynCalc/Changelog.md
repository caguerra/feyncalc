# Version 10.2.0 (December 2025)

## Removed or renamed functions, options and objects

* Renamed Commutator and AntiCommutator to FCCommutator and FCAntiCommutator to avoid clashes with new Mma 14.3 functions. Fixes issue #332 (394d50b9)
* Removed the unused Collect3. (13d86bd9)

## New functionality

* Added FCIndexDelta a placeholder for a generic Kronecker delta. (ef39f403)
* Added new routine FCClearDataTypes for removing user-defined datatype attachments. (5fe2c867)
* Added new routines FCScalarProductsSynchronizedQ and FCValuesSynchronizedQ to ensure consistency of defintions between the master kernel and the subkernels. (0161b2f6)
* Added new routine SUNFierz for applying the Fierz identity to a product of two color deltas. (d22f6e4f)
* Added new function FCLoopReplaceQuadraticEikonalPropagators for rewriting mixed quadratic-eikonal propagators in terms purely quadratic ones. (8990853d)
